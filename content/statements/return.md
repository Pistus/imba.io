# return

## Syntax

### basic return [snippet]
```imba
def method
    return true
```

### implicit return [snippet]
```imba
def method
    true
```
> If you don't explicitly return for your methods - imba will by default return the value of the last expression.

### Implicit return

If you don't explicitly return for your methods - imba will by default return the value of the last expression.
